# SaaS 着陆页质量清单

## P0（必须通过）

- [ ] 不包含 `#6366f1`（Tailwind indigo）作为主色
- [ ] Hero 区没有紫色→蓝色渐变
- [ ] 没有 emoji 作为功能图标
- [ ] 没有虚构数据（用 `—` 或占位符）
- [ ] 所有 CSS 已内联
- [ ] 全大写文本字距 ≥0.06em
- [ ] 响应式布局（移动端 < 768px）

## P1（强烈建议）

- [ ] 使用品牌色板不超过 3 种主色
- [ ] CTA 按钮有 hover 动效
- [ ] 文案无 AI 感（无"革新""赋能""颠覆"）
- [ ] 页面加载时间 < 5 秒（无大图/第三方资源）
