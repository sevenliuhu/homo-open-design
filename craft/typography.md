# 排版工艺规则

通用排版规则。设计系统决定用什么字体，本文件决定字体在不同层级如何表现。

## 字号比例尺

| 角色 | 尺寸 | 行高 | 字距 |
|---|---|---|---|
| Display | 48-72px | 1.0-1.2 | -0.02em |
| H1 | 32-48px | 1.1-1.2 | -0.01em |
| H2 | 24-32px | 1.2-1.3 | 0 |
| H3 | 20-24px | 1.3-1.4 | 0 |
| 正文 | 15-18px | 1.5-1.6 | 0 |
| 小字 | 13-14px | 1.5 | 0.01-0.02em |
| 标注 | 11-12px | 1.5 | 0.02-0.04em |

## 关键规则

1. **全大写文本字距 ≥ 0.06em** — 这是最容易被 AI 忽略的规则
2. **正文行高 1.5+** — 低于 1.5 的正文不可读
3. **Display/H1 行高 1.0-1.2** — 大标题用紧行距
4. **小字加字距** — 11-13px 时增加 0.01-0.02em 提高可读性
5. **不在 Display 上用无衬线字体** — 如果设计系统指定了衬线字体
